Usage Instructions

Learn how to use our product effectively:

  1. Open the application with administration privileges.
  2. Navigate to the desired function by clicking the appropriate button.

    • Create Dump
      1. Click the "Create Memory Dump" button.
      2. Specify the desired path for the memory dump in the input form.
      3. Click the "Create Image" button.
      4. Specify the name of the image that will be created.
      5. Choose the type of memory dump from the radio buttons.
      6. Click the "Create" button.
      7. An indicator will be provided to show the process status.

    • Analyze Dump
      1. Click the "Analyze Memory Dump" button.
      2. Choose the image dump by clicking the "Choose Files" button.
      3. Click the "Analyze Image" button.
      4. Specify the module function by clicking one or more of the checkboxes.
      5. Click the "Analyze" button.
      6. An indicator will be provided to show the process status.
      7. Process output can be view by clicking "View Output" button after all process are finished.

    • Generate Report
      1. Click the "Analyze Image" button.
      2. An indicator will be provided to show the process status.
      3. Process list will be displayed.

  3. Refer to the troubleshooting section if any error occured.
